# rax_m3u8_downloader
这是我为了学习tkinter用python 写的一个下载m3u8视频的小程序，程序使用了多线程下载，下载后自动合并成一个视频文件，方便播放。

目前的众多视频都是m3u8的播放类型，只要知道视频的m3u8地址，就可以完美下载整个视频。
测试下载链接版权原因就不公开了
